Positioning

November 8, 2021

  • Static position This is the default CSS position applicable to all HTML elements. This position place the HTML elements based on normal document flow. Using this position, top/right/bottom/left properties can not be applicable to elements (ie., static position elements donâ€™t obey top/right/bottom/left properties).
    Note: Page scrolling does affect this position.
  • Relative position This position places the element relative to its normal position. This position is relative to normal document flow. Here top/right/bottom/left properties can be applied to elements.
    Note:Page scrolling does affect this position. If position only applied without top/right/bottom/left properties, it will act like Static position.
  • Absolute position This position places the element at the exact position specified.
    Note: Page scrolling does affect this position. This position element is completely removed from normal document flow.
  • Fixed position This position places the element at a fixed place relative to the viewport. Page scrolling does not affect this position.
    Note: This position element is completely removed from normal document flow.

Margins, Border, & Padding

November 7, 2021

Margins, borders, and padding are important when formatting. The V school video explains this, but I found the picture most helpful in understanding how each one works. Basically, working from the inside out when it comes to each html element.

    CSS Properties:
  • Border:
    • Example: border: 3px solid black
    Borders need to specify: width, style, color
  • Padding:
    • Example: padding: 15px
    Padding is inside the border
  • Margin:
    • Example: margin:15px
    Margins are outside of border
    With all of these properties, you can specify: top, bottom, left, right

Live Server

November 2, 2021

Live Server is an extension is VS Code, I really like this feature because you can see your changes in real time, which is very helpful. To easly open Live Server in VS code hit: Shift + Ctrl + P, this will open your server window. It will update when you save changes within your .html & .css files. Simply hit: Ctrl + S to quickly save and changes.
